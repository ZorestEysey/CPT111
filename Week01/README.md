# Week 1: Data, Variables, A Simple Program

## File Structure

## Tasks
### Lab Exercise #1.1 Simple Math
Write a Java program that uses basic math to print computation results as in the test case below.  
Test case:  
Output:  
```
num1 + num2 = 13
num1 / num2 = 3
real1 + real2 = 13.0
real1 / real2 = 3.3333333333333335
```
### Lab Exercise #1.2 ASCII Art
Write a Java program that prints characters depicting a Christmas tree as in the test case below.  
Note that you will need to use escape characters.  
Test case:  
Output:  
```
   *
  / \
 /   \
/     \
--| |--
-------
```
### Exercise #1.1 "I love you 3000."
Write a Java program that prints "I love you 3000."  
Note that includes the two double quotes.  
Test case:  
Output: `"I love you 3000."`  
### CW1 #1.1 Smiling PC
Write a Java program that prints characters depicting a smiling PC as in the test case below.  
Note that you will need to use escape characters.  
Test Case:  
Output:  
```
  _________________
 |   ___________   |
 |  |   0   0   |  |
 |  |  )  _  (  |  |
 |  |   \___/   |  |
 |  |___________|  |
 |______|\_/|______|
   __|__|/ \|__|__
  // *********** \\
 // ************* \\
---------------------
```
## Approach
- Simple Math: Use basic arithmetic operators `+` and `/`.
- ASCII Art: Use escape characters such as `\` and `\\`
- String Output: Practice printing quotation marks using escape character `\"`

## Notes
- Each task is implemented as a separate class
- Focus on understanding basic syntax and output formatting
