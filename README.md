# Course Structure
This repository is organized by weekly lab topics:
- Week 1: Data, Variables, A Simple Program
- Week 2: Data Types, Math, Client Input, Debugging
- Week 3: Control Flow
- Week 4: Arrays, For Loops, Static Methods 1
- Week 5: Static Methods 2 and Strings
- Week 6: Introduction to Objects
- Week 8: More Objects and Inheritance
- Week 9: Objects, Inheritances and Polymorphism, and Exception Handling
- Week 10: File I/O, List, Set and Map
- Week 11: Recursion and Date
- Week 12: GUI with JavaFX
