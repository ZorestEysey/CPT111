# CPT111 Java Labs

## Course Structure

This repository is organized by weekly lab topics, covering fundamental to intermediate Java concepts:

* **Week 1** - Data, Variables, and a Simple Program
* **Week 2** – Data Types, Math, User Input, and Debugging
* **Week 3** – Control Flow
* **Week 4** – Arrays, For Loops, and Static Methods (Part 1)
* **Week 5** – Static Methods (Part 2) and Strings
* **Week 6** – Introduction to Objects
* **Week 8** – More Objects and Inheritance
* **Week 9** – Inheritance, Polymorphism, and Exception Handling
* **Week 10** – File I/O, Lists, Sets, and Maps
* **Week 11** – Recursion and Date Handling
* **Week 12** – GUI Development with JavaFX

## Description

This repository contains all lab exercises for the CPT111 (Java Programming) course.  
The labs are organized by week, following the course progression from basic syntax to more advanced topics.

## How to Use

* It is recommended to explore the code week by week
* Some exercises may depend on concepts introduced in earlier weeks
* You may place related classes in the same package when necessary

## Environment

* IDE: IntelliJ IDEA
* Java Version: JDK 17 (compatible with JDK 26)
